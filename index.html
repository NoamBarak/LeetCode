Two Sum

Wix
Problem Statement

Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target.

Example
Input:  nums = [2, 7, 11, 15], target = 9
Output: [0, 1]  // 2 + 7 == 9
The Catch

One-pass hash map: for each nums[i], look up target - nums[i] in the map before inserting nums[i]. If the complement is already there, you're done.

Solution — two-sum.cpp
two-sum.cpp
class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
        unordered_map<int,int>seen;
        for(int i=0; i<nums.size();i++)
        {
            int rem = target-nums[i];
            if(seen.count(rem))
                return {seen[rem],i};
            seen[nums[i]]=i;
        }
        return {-1,-1};
    }
};

Roman to Integer

Wix
Problem Statement

Given a Roman numeral string s, convert it to an integer. Roman numerals use subtraction when a smaller value precedes a larger one (e.g. IV = 4, IX = 9).

Examples
"III" → 3  ·  "IV" → 4  ·  "IX" → 9  ·  "MCMXCIV" → 1994
The Catch

Lookahead comparison: if value[s[i]] < value[s[i+1]], the pair is a subtraction case — add value[s[i+1]] - value[s[i]] and skip both characters with i++. Otherwise just add value[s[i]].

Solution — roman-to-integer.cpp
roman-to-integer.cpp
class Solution {
public:
    int romanToInt(string s) {
        map<char, int> m = { {'I', 1}, {'V', 5}, {'X', 10},
        {'L', 50}, {'C', 100}, {'D', 500}, {'M', 1000} };
        int res=0;
        if(s.size()==0) return 0;
        if (s.size()==1) return m[s[0]];
        for(int i=0;i<s.size(); i++){
            if(i<s.size()-1){
                if(m[s[i]]< m[s[i+1]]){
                    res+=(m[s[i+1]]-m[s[i]]);
                    i++;
                    continue;
                }
            }
            res+=m[s[i]];
        }
        return res;

    }
};

Median of Two Sorted Arrays

Wix
Problem Statement

Given two sorted arrays nums1 and nums2, return the median of the two sorted arrays. Overall time complexity must be O(log(min(m, n))).

Examples
nums1=[1,3], nums2=[2] → 2.0
nums1=[1,2], nums2=[3,4] → 2.5
The Catch

Binary-search for a partition in the smaller array. The partition in the second array is derived automatically: cut2 = (m+n+1)/2 - cut1. A valid partition satisfies l1 ≤ r2 and l2 ≤ r1. Use INT_MIN / INT_MAX as sentinels at the edges to avoid boundary checks.

Solution — median-of-two-sorted-arrays.cpp
median-of-two-sorted-arrays.cpp
class Solution {
public:
    double findMedianSortedArrays(vector<int>& nums1, vector<int>& nums2) {
        if (nums1.size() > nums2.size()) {
            return findMedianSortedArrays(nums2, nums1);
        }

        int m = nums1.size();
        int n = nums2.size();

        int left = 0, right = m;

        while (left <= right) {
            int cut1 = left + (right - left) / 2;
            int cut2 = (m + n + 1) / 2 - cut1;

            int l1 = (cut1 == 0) ? INT_MIN : nums1[cut1 - 1];
            int r1 = (cut1 == m) ? INT_MAX : nums1[cut1];

            int l2 = (cut2 == 0) ? INT_MIN : nums2[cut2 - 1];
            int r2 = (cut2 == n) ? INT_MAX : nums2[cut2];

            if (l1 <= r2 && l2 <= r1) {
                if ((m + n) % 2 == 1) {
                    return max(l1, l2);
                }
                return (max(l1, l2) + min(r1, r2)) / 2.0;
            }
            else if (l1 > r2) {
                right = cut1 - 1;
            }
            else {
                left = cut1 + 1;
            }
        }

        return 0.0;
    }
};

Valid Anagram

Wix
Problem Statement

Given two strings s and t, return true if t is an anagram of s, and false otherwise.

Examples
s="anagram", t="nagaram" → true
s="rat", t="car" → false
The Catch

Two frequency arrays of size 26 — one per string. Tally each character, then compare the arrays index by index. No sorting needed, O(n) time and O(1) space.

Solution — valid-anagram.cpp
valid-anagram.cpp
class Solution {
public:
    bool isAnagram(string s, string t) {
        vector <int> sChars(26,0);
        vector <int> tChars(26,0);
        for(int i=0; i<s.length(); i++){
            if(int(s[i]>=97)) sChars[s[i]-97]++;
            else sChars[s[i]-65]++;
        }
        for(int i=0;i<t.length();i++){
            if(int(t[i]>=97)) tChars[t[i]-97]++;
            else tChars[t[i]-65]++;
        }
        for(int i=0;i<sChars.size(); i++){
            if(sChars[i]!=tChars[i]) return false;
        }
        return true;

    }
};

Find the Peaks

Wix
Problem Statement

You are given a 0-indexed array mountain. A peak is an element strictly greater than both neighbours. Return the indices of all peaks.

Example
mountain = [1, 4, 3, 8, 5] → [1, 3]
// mountain[1]=4>{1,3}   mountain[3]=8>{3,5}
The Catch

After confirming that index i is a peak, immediately execute an extra i++ — the element at i+1 is already known to be less than mountain[i], so it cannot be a peak. This avoids a redundant comparison on the next iteration.

Solution — find-the-peaks.cpp
find-the-peaks.cpp
class Solution {
public:
    vector<int> findPeaks(vector<int>& mountain) {
        if(mountain.size()<=1) return {};
        if(mountain.size()==2){
            if(mountain[0]<mountain[1]) return {1};
            if(mountain[1]>mountain[0]) return {0};
            return {}; //same value
        }
        vector<int> res;
        for(int i=1;i<mountain.size()-1;i++){
            if(mountain[i-1]<mountain[i] && mountain[i]>mountain[i+1]){
                res.push_back(i);
                i++; //the next one is for sure NOT a peak
            }
        }
        return res;
    }
};

Pascal's Triangle

Wix
Problem Statement

Given an integer numRows, return the first numRows of Pascal's triangle. Each number is the sum of the two numbers directly above it.

Example
numRows=5 → [[1],[1,1],[1,2,1],[1,3,3,1],[1,4,6,4,1]]
The Catch

Build row by row. The rule is simple: edges are always 1; any interior cell at position (r, i) equals res[r-1][i-1] + res[r-1][i]. No formula needed — just reuse the previous row directly.

Solution — pascals-triangle.cpp
pascals-triangle.cpp
class Solution {
public:
    vector<vector<int>> generate(int numRows) {
        if(numRows==0) return {{}};
        if(numRows==1) return {{1}};
        if(numRows==2) return {{1}, {1,1}};
        vector<vector<int>> res= {{1}, {1,1}};
        for(int r=2; r<numRows; r++){
            res.push_back({});
            for(int i=0;i<=r; i++){
                if(i==0 || i==r){
                    res[r].push_back(1);
                }
                else {
                    res[r].push_back(res[r-1][i-1] + res[r-1][i]);
                }
            }
        }
        return res;
    }
};

Unique Paths

Wix
Problem Statement

A robot starts at the top-left of an m × n grid and must reach the bottom-right, moving only down or right. How many unique paths are there?

Examples
m=3, n=7 → 28  ·  m=3, n=2 → 3
The Catch

Top-down memoized DFS: f(i,j) = f(i+1,j) + f(i,j+1). Base case: f(m-1, n-1) = 1 (reached the goal); out-of-bounds = 0. Without the memo table, subproblems overlap exponentially — the cache turns it into O(m·n).

Solution — unique-paths.cpp
unique-paths.cpp
class Solution {
public:
    int uniquePaths(int m, int n) {
        vector<vector<int>> memo(m, vector<int>(n, -1));
        return f(m, n, 0, 0, memo);
    }

    int f(int m, int n, int i, int j, vector<vector<int>>& memo){
        if (i >= m || j >= n) return 0;

        if (i == m - 1 && j == n - 1) return 1;

        if (memo[i][j] != -1) return memo[i][j];

        return memo[i][j] = f(m, n, i + 1, j, memo) +
                            f(m, n, i, j + 1, memo);
    }
};

Word Search

Wix
Problem Statement

Given an m × n character grid and a word, return true if the word can be constructed from sequentially adjacent (horizontally/vertically) cells, each used at most once.

Example
board=[["A","B","C","E"],["S","F","C","S"],["A","D","E","E"]], word="ABCCED" → true
The Catch

DFS backtracking: set visited[i][j] = true before recursing into all four neighbours, then restore it to false after — this frees the cell for completely different paths. Without the restore step the algorithm would wrongly block valid routes.

Optimisation Note

If the board lacks enough of some character in word, return false immediately. Reversing the word when its last character is rarer than the first reduces the DFS branching factor.

Solution — word-search.cpp
word-search.cpp
class Solution {
public:
    bool exist(vector<vector<char>>& board, string word) {
        vector<vector<bool>> visited(board.size(), vector<bool>(board[0].size(), false));
        for(int i=0; i<board.size();i++){
            for(int j=0;j<board[0].size();j++){
                if(board[i][j]==word[0]){
                    if(func(board, word, 0, i, j, visited))
                        return true;
                }

            }
        }
        return false;
    }

    bool func(vector<vector<char>>& board, string word, int wordIdx, int i, int j, vector<vector<bool>>& visited){
        if(i>=board.size() || j>=board[0].size() || i<0 || j<0 ) return false;
        if(visited[i][j]) return false;
        if(board[i][j]!=word[wordIdx]) return false;
        if(wordIdx == word.length()-1) return true; //last char
        visited[i][j]= true;
        bool found = (func(board, word, wordIdx+1, i+1, j, visited) ||
            func(board, word, wordIdx+1, i, j+1, visited) ||
            func(board, word, wordIdx+1, i-1, j, visited) ||
            func(board, word, wordIdx+1, i, j-1, visited));
        visited[i][j]= false;
        return found;
    }
};

Word Search II

Wix
Problem Statement

Given an m × n board and a list of strings words, return all words that exist on the board. Each word uses adjacent cells without reuse.

Example
board=[["o","a","a","n"],["e","t","a","e"],["i","h","k","r"],["i","f","l","v"]]
words=["oath","pea","eat","rain"] → ["eat","oath"]
The Catch

The same DFS backtracking as Word Search (mark/unmark visited), but wrapped in an outer loop over every word. Each word gets a fresh visited grid. The optimal approach would use a Trie to search all words in one pass, but even the per-word DFS works correctly here.

Solution — word-search-ii.cpp
word-search-ii.cpp
class Solution {
public:
    vector<string> findWords(vector<vector<char>>& board, vector<string>& words) {
        vector<string> res;

        for (int w = 0; w < words.size(); w++) {
            bool isFound = false;
            string word = words[w];

            vector<vector<bool>> visited(board.size(), vector<bool>(board[0].size(), false));

            for (int i = 0; i < board.size() && !isFound; i++) {
                for (int j = 0; j < board[0].size() && !isFound; j++) {
                    if (findWord(board, word, 0, i, j, visited)) {
                        res.push_back(word);
                        isFound = true;
                    }
                }
            }
        }

        return res;
    }

    bool findWord(vector<vector<char>>& board, string word, int idx, int row, int col, vector<vector<bool>>& visited) {
        if (row < 0 || row >= board.size()) return false;
        if (col < 0 || col >= board[0].size()) return false;
        if (visited[row][col]) return false;
        if (board[row][col] != word[idx]) return false;

        if (idx == word.length() - 1) return true;

        visited[row][col] = true;

        bool found = findWord(board, word, idx + 1, row + 1, col, visited) ||
                     findWord(board, word, idx + 1, row - 1, col, visited) ||
                     findWord(board, word, idx + 1, row, col + 1, visited) ||
                     findWord(board, word, idx + 1, row, col - 1, visited);

        visited[row][col] = false;

        return found;
    }
};

Parse Lisp Expression

Wix
Problem Statement

Evaluate a Lisp-like expression supporting (let var val ... expr) with lexical scoping, (add e1 e2), and (mult e1 e2). Return the integer result.

Examples
"(let x 2 (mult x (let x 3 y 4 (add x y))))" → 14
"(add 1 2)" → 3  ·  "(let x 2 (add x 1))" → 3
The Catch

Model lexical scoping with a map from variable name → stack of values. When a let binds a variable, push the new value; when the let scope exits, pop it (rollback). This means inner scopes can shadow outer ones without destroying them — and the outermost value is always at the bottom of the stack.

Solution — parseLispExp.cpp
parseLispExp.cpp
#include <string>
#include <unordered_map>
#include <vector>
#include <cctype>
using namespace std;

class Solution {
public:
    unordered_map<string, vector<int>> scope;

    int evaluate(string expression) {
        int pos = 0;
        return parse(expression, pos);
    }

private:
    int parse(const string& s, int& pos) {
        if (s[pos] == '(') {
            ++pos; // skip '('

            if (s.compare(pos, 3, "add") == 0) {
                pos += 4; // skip "add "
                int a = parse(s, pos);
                ++pos;    // skip space
                int b = parse(s, pos);
                ++pos;    // skip ')'
                return a + b;
            }

            if (s.compare(pos, 4, "mult") == 0) {
                pos += 5; // skip "mult "
                int a = parse(s, pos);
                ++pos;    // skip space
                int b = parse(s, pos);
                ++pos;    // skip ')'
                return a * b;
            }

            // let
            pos += 4; // skip "let "
            vector<string> assigned;

            while (true) {
                if (s[pos] == '(' || s[pos] == '-' || isdigit(s[pos])) {
                    int result = parse(s, pos);
                    ++pos; // skip ')'
                    rollback(assigned);
                    return result;
                }

                string name = parseVar(s, pos);

                if (s[pos] == ')') {
                    int result = scope[name].back();
                    ++pos; // skip ')'
                    rollback(assigned);
                    return result;
                }

                ++pos; // skip space

                // If after the variable comes the final expression, not assignment
                if (s[pos] == '(' || s[pos] == '-' || isdigit(s[pos])) {
                    int val = parseAtomOrExpr(s, pos);
                    rollback(assigned);
                    return val;
                }

                // Need to distinguish between:
                // variable as final expr
                // variable followed by assignment value
                int save = pos;
                string nextToken = parseVar(s, save);

                if (save >= (int)s.size() || s[save] == ')') {
                    int result = scope[name].back();
                    rollback(assigned);
                    return result;
                }

                int val = parse(s, pos);
                scope[name].push_back(val);
                assigned.push_back(name);

                if (s[pos] == ' ')
                    ++pos;
            }
        }

        return parseAtomOrExpr(s, pos);
    }

    int parseAtomOrExpr(const string& s, int& pos) {
        if (s[pos] == '(') {
            return parse(s, pos);
        }

        if (s[pos] == '-' || isdigit(s[pos])) {
            return parseInt(s, pos);
        }

        string var = parseVar(s, pos);
        return scope[var].back();
    }

    int parseInt(const string& s, int& pos) {
        int sign = 1;
        if (s[pos] == '-') {
            sign = -1;
            ++pos;
        }

        int num = 0;
        while (pos < (int)s.size() && isdigit(s[pos])) {
            num = num * 10 + (s[pos] - '0');
            ++pos;
        }
        return sign * num;
    }

    string parseVar(const string& s, int& pos) {
        string var;
        while (pos < (int)s.size() && s[pos] != ' ' && s[pos] != ')') {
            var += s[pos++];
        }
        return var;
    }

    void rollback(const vector<string>& assigned) {
        for (const string& var : assigned) {
            scope[var].pop_back();
        }
    }
};

Fibonacci Number

Wix
Problem Statement

Given n, return F(n) where F(0)=0, F(1)=1, and F(n) = F(n-1) + F(n-2).

Examples
F(0)=0 · F(1)=1 · F(5)=5 · F(10)=55
The Catch

Only the last two values matter — no array, no recursion. Keep prev2 and prev1, compute curr = prev1 + prev2, then slide: prev2 = prev1; prev1 = curr. O(n) time, O(1) space.

Solution — fibo.cpp
fibo.cpp
int fib(int n) {
    if (n <= 1) return n;

    int prev2 = 0;
    int prev1 = 1;

    for (int i = 2; i <= n; i++) {
        int curr = prev1 + prev2;
        prev2 = prev1;
        prev1 = curr;
    }

    return prev1;
}

DFS Templates

Wix
Reference Patterns

Three reusable DFS patterns: recursive traversal, iterative traversal with an explicit stack, and a backtracking template with visited-state management.

The Catch

The cardinal backtracking rule: unmark visited after the recursive call returns — not before. This is what distinguishes backtracking from plain DFS. Unmarking restores the state so that sibling branches of the search tree can explore the same node via a different path.

Templates — dfs.cpp
dfs.cpp
// Recursive DFS
void dfs(Node* node) {
    if (!node) return;

    visit(node);

    dfs(node->left);
    dfs(node->right);
}

//-------------------------------------------------------
// Iterative DFS
stack<Node*> st;
st.push(root);

while (!st.empty()) {
    Node* curr = st.top();
    st.pop();

    visit(curr);

    if (curr->right) st.push(curr->right);
    if (curr->left) st.push(curr->left);
}

//-------------------------------------------------------
// DFS template
bool dfs(...) {
    if (/*stop condition*/)
        return;

    mark visited;

    for (each neighbor) {
         if (neighbor is not visited)
             dfs(...);
    }

    unmark visited; // backtrack
}

Reverse Linked List

IAI
Problem Statement

Given the head of a singly linked list, reverse the list in-place and return the new head.

Examples
[1,2,3,4,5] → [5,4,3,2,1]
[1,2] → [2,1]
The Catch

Three-pointer slide: always save p3 = p2->next before overwriting p2->next = p1 — otherwise you lose the rest of the list. Then advance: p1 = p2; p2 = p3; p3 = p3->next. Single pass, O(n) time, O(1) space.

Solution — reverse-linked-list.cpp
reverse-linked-list.cpp
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */
class Solution {
public:
    ListNode* reverseList(ListNode* head) {
        if(head==nullptr) return nullptr;
        if(head->next == nullptr) return head;
        ListNode* p1 = head;
        ListNode* p2 = p1->next;
        ListNode* p3 = p2->next;
        p1->next=nullptr;
        while(p2!=nullptr){
            p2->next = p1;
            p1 = p2;
            p2 = p3;
            if (p3!=nullptr) p3 = p3->next;
        }
        return p1;

    }
};

Meeting Rooms III

IAI
Problem Statement

Given n rooms (0-indexed) and sorted meetings [start, end], assign each meeting to the lowest-numbered free room. If no room is free, delay the meeting to the earliest available room (same duration). Return the room that hosted the most meetings.

Examples
n=2, meetings=[[0,10],[1,5],[2,7],[3,4]] → 0
n=3, meetings=[[1,20],[2,10],[3,5],[4,9],[6,8]] → 1
The Catch

Two min-heaps: availableRooms ordered by room number, and busyRooms ordered by end-time (tie-break: smaller room). Before each meeting, drain busyRooms into availableRooms for all rooms whose endTime ≤ start. If availableRooms is still empty, pop the earliest-ending room from busyRooms and push it back with newEnd = freeTime + duration.

Solution — meeting-rooms-iii.cpp
meeting-rooms-iii.cpp
#include <iostream>
#include <vector>
#include <queue>
#include <algorithm>

using namespace std;

class Solution {
public:
    int mostBooked(int n, vector<vector<int>>& meetings) {
        sort(meetings.begin(), meetings.end());

        // Free rooms: smallest room number first
        priority_queue<int, vector<int>, greater<int>> availableRooms;
        for (int room = 0; room < n; ++room) {
            availableRooms.push(room);
        }

        // Busy rooms: {endTime, roomNumber}
        // earliest end time first, and if tie then smaller room number first
        priority_queue<
            pair<long long, int>,
            vector<pair<long long, int>>,
            greater<pair<long long, int>>
        > busyRooms;

        vector<long long> meetingCount(n, 0);

        for (const auto& meeting : meetings) {
            long long start = meeting[0];
            long long end = meeting[1];
            long long duration = end - start;
            cout<< "start: "<< start<< ", end: "<<end<<endl;

            // Release every room that became free by meeting start time
            while (!busyRooms.empty() && busyRooms.top().first <= start) {
                int freedRoom = busyRooms.top().second;
                cout<< "freedRoom: " << freedRoom <<endl;
                busyRooms.pop();
                availableRooms.push(freedRoom);
            }

            if (!availableRooms.empty()) {
                // Use the smallest available room
                int room = availableRooms.top();
                cout<< "available room: " << room << endl;
                availableRooms.pop();

                busyRooms.push({end, room});
                meetingCount[room]++;
            } else {
                cout<< "no room is free. ";
                // No room is free: delay meeting to earliest available room
                auto [freeTime, room] = busyRooms.top();
                busyRooms.pop();

                long long newEnd = freeTime + duration;
                busyRooms.push({newEnd, room});
                meetingCount[room]++;
                cout<< " New end: " << newEnd <<endl;

            }
        }

        int answer = 0;
        for (int room = 1; room < n; ++room) {
            if (meetingCount[room] > meetingCount[answer]) {
                answer = room;
            }
        }

        return answer;
    }
};

Valid Parenthesis String

IAI
Problem Statement

Given a string s containing only '(', ')', and '*', return true if it is valid. A '*' can be treated as '(', ')', or an empty string.

Examples
s = "(*)" → true
s = "(*))" → true
s = "(*())" → true
The Catch

Use two stacks of indices — one for '(' and one for '*'. On encountering ')', prefer to pop an unmatched '('; fall back to a '*'; otherwise return false. After the scan, match any remaining '(' with '*' — but only if star.top() > open.top(), because a '*' can only substitute a '(' that appears before it.

Solution — valid-parenthesis-string.cpp
valid-parenthesis-string.cpp
class Solution {
public:
    bool checkValidString(string s) {
        stack<int> open;
        stack<int> star;

        for (int i = 0; i < s.size(); i++) {
            if (s[i] == '(') {
                open.push(i);
            } else if (s[i] == '*') {
                star.push(i);
            } else { // ')'
                if (!open.empty()) {
                    open.pop();
                } else if (!star.empty()) {
                    star.pop();
                } else {
                    return false;
                }
            }
        }

        while (!open.empty() && !star.empty()) {
            if (open.top() > star.top()) return false; // star is before '('
            open.pop();
            star.pop();
        }

        return open.empty();
    }
};

Binary Search

IAI
Problem Statement

Given a sorted array nums and a target, return the index of target, or -1 if it does not exist. Must run in O(log n).

Example
nums=[1,3,5,7,9,11], target=7 → index 3
The Catch

Two subtle rules: (1) compute mid = left + (right - left) / 2 instead of (left + right) / 2 to prevent integer overflow. (2) Always move to mid + 1 or mid - 1 — never just mid — so the search range shrinks every iteration and the loop is guaranteed to terminate.

Solution — binary-search.cpp
binary-search.cpp
#include <iostream>
#include <vector>
using namespace std;

int binarySearch(const vector<int>& nums, int target) {
    int left = 0, right = nums.size() - 1;

    while (left <= right) {
        int mid = left + (right - left) / 2;  // avoid overflow

        if (nums[mid] == target)
            return mid;
        else if (nums[mid] < target)
            left = mid + 1;
        else
            right = mid - 1;
    }
    return -1;
}

int main() {
    vector<int> nums = {1, 3, 5, 7, 9, 11};
    int target = 7;

    int result = binarySearch(nums, target);

    if (result != -1)
        cout << "Found at index: " << result << endl;
    else
        cout << "Not found" << endl;

    return 0;
}

Valid Parentheses

IAI
Problem Statement

Given a string s containing only '(', ')', '{', '}', '[', ']', determine if the input string is valid. Every opening bracket must be closed by the same type in the correct order.

Examples
"()" → true  ·  "()[]{}" → true  ·  "([)]" → false  ·  "{[]}" → true
The Catch

Stack of opening brackets: push any opener. On a closer, immediately check whether the stack is empty or whether stack.top() is the matching opener — if either condition fails, return false. Pop on success. At the end, a valid string leaves an empty stack.

Solution — valid-parentheses.cpp
valid-parentheses.cpp
class Solution {
public:
    bool isValid(string s) {
        stack<char>paren;
        for(int i=0;i<s.length();i++){
            if(s[i]=='(' || s[i]=='[' || s[i]=='{'){
                paren.push(s[i]);
            }
            else{
                if(paren.empty()) return false;
                if(s[i]==')' && paren.top()!='(') return false;
                if(s[i]==']' && paren.top()!='[') return false;
                if(s[i]=='}' && paren.top()!='{') return false;
                paren.pop();
            }
        }
        if(paren.empty()) return true;
        return false;
    }
};

Generate Parentheses

IAI
Problem Statement

Given n pairs of parentheses, write a function to generate all combinations of well-formed parentheses.

Example
n=3 → ["((()))","(()())","(())()","()(())","()()()"]
The Catch

Backtracking with two counters (open, close): add '(' only when open < n, add ')' only when close < open. These two constraints alone guarantee every fully-built string is already valid — no post-validation step needed. When cur.length() == 2*n, record the result.

Solution — generate-parentheses.cpp
generate-parentheses.cpp
class Solution {
public:
    vector<string> generateParenthesis(int n) {
        vector<string> res;
        func(0, 0, n, "", res);
        return res;
    }
    void func(int open, int close, int n, string cur, vector<string>& result) {
        if (cur.length() == 2 * n) {
            result.push_back(cur);
            return;
        }

        if (open < n) {
            func(open + 1, close, n, cur + '(', result);
        }

        if (close < open) {
            func(open, close + 1, n, cur + ')', result);
        }
    }
};
